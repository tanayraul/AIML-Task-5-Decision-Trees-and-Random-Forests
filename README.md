# AIML-Task-5-Decision-Trees-and-Random-Forests
Learn tree-based models for classification &amp; regression. 

This project focuses on implementing and understanding tree-based machine learning algorithms for classification using the Heart Disease dataset. The primary objective was to predict whether a patient is likely to have heart disease based on medical attributes such as age, chest pain type, cholesterol level, blood pressure, heart rate, and other clinical indicators. The project was developed in Google Colab using Python, Pandas, Matplotlib, and Scikit-learn. Decision Tree Classifier and Random Forest Classifier were used to learn classification rules from the dataset, while visualization techniques were applied to understand model behavior and feature importance. The workflow included dataset loading, preprocessing, train-test splitting, model training, evaluation, tree visualization, and performance comparison.

The Decision Tree model was first trained to understand how individual decision rules classify patients into heart disease and non-heart disease categories. The generated tree structure was visualized to observe how features contribute to predictions and how model complexity affects performance. Overfitting analysis was performed by comparing training and testing accuracy across different tree depths, demonstrating the importance of controlling model complexity using the max_depth parameter. A Random Forest model was then implemented, which combines multiple decision trees to improve prediction stability and reduce overfitting. Feature importance analysis was conducted to identify the most influential medical attributes affecting disease prediction, providing valuable insights into which clinical factors contribute most significantly to classification decisions.

The results showed that Random Forest generally achieved higher and more stable accuracy than a single Decision Tree because of its ensemble learning approach. Cross-validation was used to evaluate model consistency across multiple data splits, providing a more reliable estimate of real-world performance than a single train-test evaluation. Important features such as chest pain type, maximum heart rate achieved, oldpeak values, thalassemia information, and major vessel count were found to have strong predictive power. Overall, the project demonstrates practical applications of tree-based machine learning models, model optimization techniques, ensemble learning, feature interpretation, and performance evaluation methods commonly used in healthcare analytics and predictive modeling tasks.
